# AI Email Summarizer

## 📌 Overview
AI-powered email summarization system built using n8n automation and Google Gemini AI.  
This workflow automatically fetches recent emails from Gmail, summarizes them using Gemini LLM, and generates a structured summary report.

---

## 🚀 Features
- Automated email fetching from Gmail
- AI-based summarization using Google Gemini
- Scheduled workflow execution
- Summary aggregation for clarity
- Automatic draft generation in Gmail

---

## 🛠 Tech Stack
- n8n (Workflow Automation)
- Google Gemini LLM
- Gmail API
- JSON-based workflow configuration

---

## 📂 Project Structure
- `Email Summary Agent.json` – n8n workflow configuration
- `README.md` – Project documentation

---

## 💡 Use Case
Helps professionals save time by automatically generating concise summaries of daily emails.
